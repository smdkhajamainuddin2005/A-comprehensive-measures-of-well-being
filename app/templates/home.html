HDI Predictor

Human Development Index Machine Learning System

Project Overview

The Human Development Index (HDI) is a summary measure of average achievement in key dimensions of human development: a long and healthy life, being knowledgeable, and having a decent standard of living.

This system uses a trained Linear Regression Model to predict a country's HDI score based on three core indicators. Enter the details below to estimate the score.
